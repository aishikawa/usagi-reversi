# Usagi-reversi
This is a reversi software using [ushi-mctslib](https://github.com/aishikawa/ushi-mctslib).

## Usage

### Build
Use ant.
```shell
$ ant
```
Abone command creates `usagi_reversi.jar` in `dest` directory.

### Run
```shell
$ java -jar dest/usagi_reversi.jar
```
